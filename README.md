# Food Delivery System

A microservices-based food delivery application built with **Spring Boot**, **Spring WebFlux**, **MongoDB**, **React**, and **Eureka Service Discovery**.

This project includes:

- **Discovery Server** for service registration
- **Food Service** as the main reactive backend microservice
- **Gateway Service** as the API layer for the React frontend
- **React Frontend** for browsing restaurants, viewing menus, adding items to cart, and placing orders
